What is CSS?

CSS (Cascading Style Sheets) allows you to create great-looking web pages

Using CSS, you can control exactly how HTML elements look in the browser, presenting your documents to your users with whatever design and layout you like. CSS can be used for text styling — for example, for changing the color and size of headings and links. It can be used to create a layout — for example, turning a single column of text into a layout with a main content area and a sidebar for related information. It can even be used for effects such as animation.
